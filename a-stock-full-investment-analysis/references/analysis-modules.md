# 16大模块分析指令与评分规则

> 每个模块的输入数据来自 Step 2 数据整理表。评分均为 1-10 分制，最终加权后映射到100分。  
> **禁止**在无数据支撑时推测评分。若关键数据缺失，基础分 5.0，并在发现项中说明。

---

## 评分基准

| 分值 | 星级 | 含义 |
|------|------|------|
| 9-10 | ★★★★★ | 极强 |
| 8-9 | ★★★★ | 强 |
| 7-8 | ★★★ | 偏强 |
| 5-7 | ★★★ | 中性 |
| 3-5 | ★★ | 偏弱 |
| 1-3 | ★ | 弱 |

## M01：公司基础与业务拆解

**数据来源**：S1-C（东方财富基础信息）→ S2（AKShare stock_individual_info_em）→ S6-B

**分析要点**：
1. 列出公司全称、股票代码、上市时间、上市板块
2. 所属行业（申万行业分类）
3. 总市值与流通市值（来自采集数据）
4. 主营业务描述（原文，100字以内概述）
5. 企业类型判断（成长型/防御型/周期型/普通型）

**评分规则**：
- 基础分：5.0
- 有详细主营业务描述：+1.0
- 有行业归类：+1.0
- 总市值>500亿（蓝筹）：+2.0；100-500亿：+1.0；<30亿：-1.0
- 上市10年以上（成熟企业）：+0.5
- ST/退市风险标的：-2.0

**输出模板**：
```
【公司基础信息】
- 公司：{全称}（{代码}）上市于{上市时间}，{板块}
- 行业：{申万行业}，企业类型：{成长型/防御型/周期型}
- 市值：总市值{X}亿，流通市值{X}亿
- 主营：{100字以内主营业务描述}
- 规模判断：{大市值蓝筹/中市值/小市值}
```

## M02：财务质量深度核验

**数据来源**：S1-C → S2（财务指标+利润表+资产负债表+现金流量表）→ S6-B

**分析要点**（必须使用采集到的真实数值）：

**① 盈利能力**
- ROE：采集值____%
  - ≥20%：+2.0，极强盈利能力
  - 15-20%：+1.5，良好
  - 10-15%：+0.5，尚可
  - <5%：-1.0，偏弱
- 毛利率：采集值____%（>40%：+1.0 高护城河；<10%：-0.5 同质化）
- 净利率：采集值____%

**② 成长性**
- 近一期营收同比：采集值____%
- 近一期净利润同比：采集值____%
  - >30%：+1.5，高增长
  - 10-30%：+0.5，稳健增长
  - <-20%：-1.5，🚨业绩恶化

**③ 近3-4年年度趋势**（列表格式输出，必须填真实数值）：
| 年份 | 营收（亿） | 净利润（亿） | 毛利率 | 营收同比 | 利润同比 |
|------|-----------|------------|--------|---------|---------|

**④ 财务安全性**
- 资产负债率：采集值____%
  - >70%：-1.5，🚨财务杠杆过高
  - 60-70%：-0.5，偏高
  - <40%：+0.5，健康
- 经营现金流/净利润：采集值___x
  - ≥1.2：+1.0，利润含金量高
  - <0.5：-1.0，⚠️盈利质量存疑

**⑤ 风险排雷**
- 商誉/总资产：若>30%：-1.0，🚨减值风险
- 扣非/净利润：若<70%：-1.0，⚠️非经常性损益虚增

**评分范围**：基础分5.0，依上述规则调整，最终1.0-10.0。

**输出模板**：
```
【财务核心数据】（数据来自：{数据源}，报告期：{YYYY年Q/年报}）
- 盈利能力：ROE {X}%，毛利率 {X}%，净利率 {X}%
- 最新期业绩：营收 {X}亿（同比{+/-X}%），净利润 {X}亿（同比{+/-X}%）
- 近3年营收：{YYYY年}{X}亿、{YYYY年}{X}亿、{YYYY年}{X}亿
- 财务安全：负债率 {X}%，现金流/净利润 {X}x
- 质量评级：{真成长/稳健盈利/普通质量/财务隐患}
```

**数据来源**：S1-E → S2（股东相关函数）→ S6-D

  - >40%：-1.5，🚨质押爆仓风险
  - 20-40%：-0.5，⚠️关注

**评分规则**：基础分5.0，依上述调整，1.0-10.0。

## M04：行业周期与产业链景气度

**数据来源**：S1-H → S2（stock_board_industry_cons_em）→ S6-C

**分析要点**：

**① 行业类型判断**
- 成长赛道（AI/新能源/半导体/机器人/低空经济等）：+1.5，政策红利
- 防御消费（白酒/医药/公用事业/食品饮料）：+0.5，抗周期
- 强周期（钢铁/煤炭/化工/房地产）：注明周期位置
- 均衡型：按景气度判断

**② 行业景气度（基于今日数据）**
- 行业成分股今日平均涨跌幅：采集值____%
  - >2%：+1.5，景气上行
  - 0-2%：+0.5，温和向好
  - <-2%：-1.0，景气下行
- 行业平均PE：采集值____x（若>100，估值偏贵；若<15，价值洼地+0.5）
- 行业平均PB：采集值____x

**③ 个股相对行业**
- 个股PE vs 行业中位数PE
  - 折价>20%：+0.5（行业内低估）
  - 溢价>50%：⚠️估值明显偏贵

**④ 大盘情绪影响**
- 今日市场整体涨跌家数：采集值
- 情绪温度（上涨家数/总家数）：>60%：+0.5；<40%：-0.5

**输出模板**：
```
【行业分析】（数据来源：{数据源}）
- 所属行业：{行业名称}（{成长赛道/防御消费/强周期/均衡型}）
- 行业今日：{N}只股涨{N}跌，平均涨幅{+/-X}%，景气度：{景气上行/整理/下行}
- 行业估值：平均PE {X}x，中位数PE {X}x，个股PE {X}x（{折价/溢价}{X}%）
- 行业政策：{政策支持/中性/压制}
```

## M05：估值定价与护城河壁垒

**数据来源**：S1-A + S1-C → S2（财务指标）→ S6-B

**分析要点**：

**① 相对估值**
- PE-TTM：采集值____x
  - 对比行业均值判断高低
  - 对比历史PE分位（若可获取）
- PB：采集值____x（PB<1 为破净，+1.5）
- PS（市销率）：适用于低利润高营收行业

**② 估值合理区间（必须基于采集数据计算）**
- 合理PE区间 = 行业平均PE × 0.8 ~ 1.2
- 对应股价区间 = 合理PE × EPS

**③ 护城河判断**（定性分析）
- 品牌护城河（高端消费/医药）
- 成本护城河（规模效应/资源垄断）
- 网络效应（互联网/平台）
- 切换成本（企业软件/专用设备）
- 无护城河（同质化产品）

**④ DCF估值（简化版）**
- 当前股价 vs 采集的每股净资产 → 计算PB
- 股息率 vs 10年期国债收益率（约2.0-2.3%）
  - 股息率 > 3%：+0.5，配置价值高
  - 股息率 > 国债 × 2：+1.0

**评分规则**：基础分5.0，PB<1：+1.5；PE处于行业低分位：+1.0；PE>行业50%以上：-1.0；有护城河：+1.0-2.0。

**输出模板**：
```
【估值分析】（数据来源：{数据源}）
- 估值：PE-TTM {X}x（行业均值{X}x，{折价/溢价}{X}%），PB {X}x
- 股息率：{X}%（10年国债约2.1%，{高于/低于}国债{X}倍）
- 每股净资产：{X}元（当前价格PB={X}x）
- 合理价格区间：{X}~{X}元（基于行业PE区间×EPS）
- 护城河：{品牌/成本/网络/切换成本/无}
- 估值结论：{低估/合理/略贵/高估}
```
